## 👤 한 줄 소개
문제 해결과 실제 협업 경험을 통해 사용자를 위한 서비스를 만드는 성장지향형 프론트엔드 개발자입니다.

- **목표**: React/TypeScript/Next.js 기반으로 사용자 경험이 뛰어난 웹 서비스를 만드는 프론트엔드 개발자가 되고 싶습니다.  
- **진행**: 평일 본업이 끝난 후, 하루 2시간씩 꾸준히 JS → React → TypeScript → Next.js 학습 및 실전 프로젝트·코딩테스트 풀이로 꾸준하게 성장 중입니다.  
- **관심사**: 웹 접근성(A11y), 성능 최적화(LCP/이미지 최적화), 문서화·협업(PR/README)  
- **지향점**: 기술과 소통, 그리고 사용자를 모두 아우르는 개발자.  
  지속적인 자기계발과 실전 경험, 유연한 커뮤니케이션으로 팀에 긍정적 에너지를 더하는 사람이 되고 싶습니다.
